# 🌐 Language Translation Tool

## Project Description
A web-based Language Translation Tool built with Python and Streamlit.
It translates text between 19 languages instantly using the Google Translate API.

## Features
- Supports 19 languages including Yoruba, Hausa, Igbo and Swahili
- Auto-detects source language
- Clean and simple web interface
- Copy translated text instantly

## Technologies Used
- Python 3.x
- Streamlit
- deep-translator

## How to Run
1. Install requirements:
   pip install -r requirements.txt
2. Run the app:
   streamlit run app.py
3. Open browser at:
   http://localhost:8501
